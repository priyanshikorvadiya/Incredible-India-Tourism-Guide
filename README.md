# 🇮🇳 Incredible India Web Showcase

An immersive, responsive tourism and cultural showcase web application built to celebrate the heritage, landmarks, and diversity of India. This project utilizes modern web layout methodologies along with a mobile-first UI framework.

---

## 🎨 Tech Stack & Architecture

* **Semantic HTML5:** Structured with clear document outlines (`<section>`, `<article>`, `<header>`) to optimize SEO and accessibility.
* **Custom CSS3:** Applied custom keyframe animations, typography styling, and fallback styles for standard layout rules.
* **Bootstrap 5 Integration:** * Implemented the **Bootstrap Grid System** (Rows, Columns, and Flex utilities) for complete responsiveness across mobile, tablet, and desktop viewports.
  * Utilized pre-built component structures including modern navbar systems, image card layouts, and responsive modals.

---

## 🗂️ Project Structure

* `index.html` - The core entry point containing the landing view, destination sections, and markup.
* `style.css` - Custom style sheets for unique color palettes, layout tweaks, and hover states.
* `assets/` - Directory containing optimized project imagery, icons, and graphic resources.
